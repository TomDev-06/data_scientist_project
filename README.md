# Ruby Project - Data Scientist

This project contains two Ruby scripts to analyze data, as part of an exercise for The Hacking Project.

## Content

*   `lib/00_journalists.rb`: A script that analyzes a list of journalist Twitter handles.
*   `lib/01_cryptocurrencies.rb`: A script that analyzes a list of cryptocurrencies and their prices.=ECHEC

## Usage

To run a program, navigate to the root of the folder and use the `ruby` command followed by the file path.

```bash
ruby lib/00_journalists.rb
```
